# MX-5 Tracker

A beautiful and sleek mobile app to track your MX-5's annual mileage limit of 10,000 km.

## Features

- **Dashboard**: View current mileage, daily targets, and variance from target
- **Statistics**: Track daily average, projected annual total, and remaining budget
- **Progress Visualization**: Beautiful progress bar showing current usage
- **History**: View all your mileage entries with dates and notes
- **Easy Entry**: Quick and simple mileage entry form

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Start the development server:
```bash
pnpm expo start
```

3. Run on your device:
- Press `i` for iOS simulator
- Press `a` for Android emulator
- Scan QR code with Expo Go app on your phone

## Design

The app features a sleek, modern design with:
- Dark navy background (#1D3557)
- Vibrant red accent (#E63946) matching the MX-5's sporty character
- Clean, card-based UI with smooth animations
- Easy-to-read statistics and progress indicators
