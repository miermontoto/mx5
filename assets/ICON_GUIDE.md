# MX-5 Tracker Icon Design Guide

## Design Principles
- Minimalist design
- Dark background (#050505)
- Soul Red accent (#CC0000)
- Simple geometric shapes

## icon.png (1024x1024)
- Background: Solid #050505
- Center element: Simple red circle (Soul Red #CC0000)
- Inside circle: White "5" in clean, modern font
- Border: Thin #333333 stroke

## adaptive-icon.png (1024x1024)
- Same as icon.png but with transparent background
- Only the red circle with "5" inside
- No border

## splash-icon.png (1242x2688)
- Background: Solid #050505
- Center: Same red circle with "5"
- Below circle: "MX-5 TRACKER" text in #666666
- Keep it centered and minimal

## favicon.png (32x32)
- Background: #050505
- Simple red square with rounded corners
- White "5" inside

## Color Palette
- Background: #050505
- Primary: #CC0000 (Soul Red)
- Text: #FFFFFF
- Secondary Text: #666666
- Border: #333333

## Font Recommendations
- Use system fonts or Arial/Helvetica
- Light weight for headers
- Regular for body text
- Bold for the "5" in icons
